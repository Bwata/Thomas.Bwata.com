Instructions

Write a webpage that displays the number of patients for each blood type. First, you need to copy the file patients.txt to your server. You should look through this file in an editor. There are 1000 records (separated by end-of-line marks) representing 1000 patients. Each patient has a patient number, gender, weight, blood type and indicator for a virus (this is not real data).

You will write two webpages for this lab. The first will have a form to allow the user to enter their preferences for the report. The form will have two text boxes to allow them to enter a minimum weight and a maximum weight. It will also have two radio buttons to allow them to specify male or female. Finally there should also be a submit button.

You need to write the JavaScript code to verify the form. They must choose one of the genders and put in numbers into the text boxes such that the maximum is larger than the minimum. When they click on the submit button if there are any errors, you should display a message and allow them to enter again (this mean stopping the default action of calling the php script).

If there are no errors, it should call the php script. The script will return a page with the counts like in the image to the right. First it should read in the data from the file but only use the records that were desired (weight should be between the min and max and the gender should match the one selected from the form). For the matching records, maintain the counts for whether or not the patient has the virus and for the different blood types. You should ignore the plus/minus part of the blood type.

After you have collected all of the counts display them in a table like the one shown to the right. The counts listed there are for males between 140 and 200 lbs (just in case you want to check your work.
