CS 371 Web-Based Programming Winter 2014

Homework 1 - XHTML and CSS

Objectives

  • Learn about XHTML
  • Use structural tags
  • Gain familiarity with CSS
  • learn to use a variety of selectors
  • learn to use a variety of descriptions
  • get familiar with the box model

Instructions

For homework 1, you will be writing a web page about yourself. If you are self conscious, you can fill in parts with random junk but I would prefer that you put as much of yourself into this as you can. I will be grading the techical aspects of the page but having a description of you will help me to get to know you.

To start, put a heading with your name at top. Then place a picture of yourself (it can be silly but again it would be nice to have something I can recognize) so that it appears on the right side of the page (use CSS float). Then have one or more paragraphs that float around the picture to the left.

Next, create a table with no borders for some content. The table is to have two rows and three columns (six cells). In each cell place something different (see below). The content of each cell must be within a div and the divs must have one or more classes assigned to them. The odd ones should have a class of "odd" and the even ones must have a class of "even". Additional classes may be assigned below.

  • for cell 1 put in a unorderd list of favorite things to do.
  • in cell 2, place a picture of a person that is inspiring to you (sports, entertainment, politics, etc.)
  • place an ordered list in cell 3 of some of the things you do in a day. The list should be lettered a, b, c, etc.
  • in cell 4 place a table with a border of some of the courses you have taken (number, title, credits)
  • in cell 5 enter some hyperlinks to some of your favorite web sites. The div for this cell should also have a class of "links".
  • in cell 6 put in a few of your favorite latin quotations. For you most favorite, give it a class of "best".

You must also write the CSS (in a separate file) to:

  • make the body of page have the font times, medium, black
  • for paragraphs within tables set the font size to small
  • unordered lists must be in blue
  • text with a paragraph with a class of "best" is italicized
  • hyperlinks are to be in red and turn green after having been visited
  • margin of table should be 2px
  • margin of table within table should be 1px
  • div with a class of "odd" should have gray background
  • div with a class of "links" should have font family be sans-serif
