CS 371 Web-Based Programming Winter 2014

Homework 5 - PHP basics

Objectives

  • get more familiar with php
  • work with database functions

Instructions

You are to write a number of php functions to create and update a database table. These are separate functions that are not designed to work together as a page but just something to give you practice in writing php code. Specifically you will write

  • a function to create a table called friends that will have an id number (int auto-increment), name (string), phone (string) and age(int).
  • a function to read in a text file and populate the table with the data in the file
  • a function that allows the user to type name, phone and age into a form and on pressing submit, will add the data to the friends table
  • a function to read through the table and display the friends in an html table (include all fields - even id)

Your main page should have a links to the scripts that create the database table, that populates it, and that displays the records. There should also be a form that calls the script that adds the data from the form to the table.

Some questions you should ask yourself (but you do not need to send me the answers):

  • what happens when you create the table a second time? What happens when you create it a second time after populating it?
  • what happens when you populate the table twice?
  • if you manually delete all records what id is assigned to the next one that is inserted?
