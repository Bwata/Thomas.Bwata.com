CS 371 Web-Based Programming Winter 2014

Lab 7 - AJAX

Objectives

  • write php scripts to service AJAX requests
  • write AJAX javaScript functions to retrieve and manipulate data

Instructions

In this lab you will be using AJAX to create a webpage that automatically loads a select pull-down box and refreshes the list of clients displayed whenever a new state is chosen.

First, you need to load the data from this file, which is a tab delimited file of client data (the first line is a description of the fields which you can remove after you have noted the field names). Load these records into a table called client in your database.

Next you need to create two php scripts. The first one returns a unique list of states. You can simply return a tab or comma delimited text string. The other returns a list of clients from a given (parameter) state. This should be in an XML format and reference an XSL file (that you create) that transforms it into a nice html table with first and last name, address and phone number. For both of these, you should use an appropriate SQL query restrict the number of records rather than iterating through the entire table.

Finally, write an html file with javaScript. The first function executes on load to (AJAX) call the first php script and populate a select tag with the state entries. The other is executed whenever the user changes the value of the select control. When executed it should make an AJAX call to the second script and place the returned data into a div on the page.
