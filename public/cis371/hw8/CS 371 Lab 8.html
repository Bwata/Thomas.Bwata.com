Objectives

  • learn some typical jQuery functions

Instructions

You are to re-write lab 1 with the following changes:

  • In lab 1, you were instructed to organize the content using an html table. For this lab, change the table, to a tabbed box (tabs). In lab 1 where you had 6 things in the 6 cells of the table, for this lab, use the same content but have them on 6 different tabs.
  • Create two different classes (in the css file) for your unordered list(s). Then, whenever the user hovers over an item of the list, change the class and change it back to the original when they stop hovering. The effect should look like the item that is selected (hovered over) is emphasized.
  • Do some animation with the image (only one if you have many). For example, have it fade in and out when the user hovers and un-hovers. Or have it fade in when the tab is selected.
