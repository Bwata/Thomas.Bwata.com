Homework 6 - Data Encoding

Objectives

  • gain an appreciation for JSON
  • practice using XML and XSL

Instructions

In this lab you are to read in a file that is encoded in JSON, convert it to XML encoding and deliver it to the client via XSL. It is not something that you would ordinarily do but it allows you to get some experience with many of the concepts from this week.

The file contains information on the companies listed in the Fortune 500 from 2009. Your php script will read in this file as a single string and convert it to an object (see json_decode). Next, the script will create a DOMDocument and using the DOM methods, populate the document with the data from the object. You must also add a link to an XSL document that you create separately (by appending a text element to the DOMDocument). Finally, you need to output (echo) the xml document (using saveHTML).

To format the XML data nicely for the client, you will also need to write an XSL stylesheet (that the XML document links to - see above). This style sheet should select the top 30 companies and list the rank, the name of the company, and revenue in an html table (like the one below).
