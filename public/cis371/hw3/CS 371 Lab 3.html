CS 371 Web-Based Programming Winter 2014

Lab 3 - Dynamic JavaScript

Objectives

  • practice with JavaScript events
  • practice with using the DOM in JavaScript

Instructions

Write a webpage that allows the user to enter an order. You can use whatever you want for the product (pizza, cars, electronics, etc.), just make sure that you several different types of products with different prices. Pattern your page after the example to the right. You can make it as pretty as you want just make sure that it is readable.

To start your table with the orders should be empty except for one row that has the text boxes for item number and quantity. After the user enters a product (item number and quantity) and presses the enter button, your script should add it to the table (second to last row). So the first item goes in the first row, second one goes in the second row and so on. The text boxes and button should always appear in the last row.

Write functions that allow only numbers to be put into the text boxes. When the user presses the enter button, you should execute a function that finds the price from the price table, multiplies it by the quantity and then adds the order to the table.

Also, write the answer (in a comment) on your page to the following question: if such a web page actually existed, how could you easily hack it so that you could enter orders with a lower price?
